jquery-collapsible-fieldset
===========================

##Overview
jQuery plugin for Drupal-style collapsible fieldsets.

##Usage

````javascript
$("fieldset.collapsible").collapsible();
````

````html
<form>
	<fieldset class="collapsible">
		<legend>Fieldset Collapsible Legend</legend>
		<div>
        		<input type="text" />
		</div>
	</fieldset>
</form>
````
